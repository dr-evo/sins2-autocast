Community autocast enhancements for vanilla.

Modifies simulation - all players in multiplayer must have this mod.

Built for patch 1.45 (Eivonn's fine) , 2025-10-06+

# Fixes
## Advent
- halcyon carrier - shield blessing - overlap - avoid casting if another shield blessing is active

## TEC
- flak burst - overlap - avoid casting if another flak burst is active in its area
- radiation bomb - don't cast unless you have at least 5 non-affected targets

## Vasari
- skirantra carrier - replicate forces - prioritization - target only combat units with a weapon (TODO prioritize cruisers over frigates)
- jarrasul evacuator - subspace rupture - prioritization, aoe -  prioritize targets affected with gravity warhead, use only if you have at least 5 valid targets
- jarrasul evacuator - gravity warhead - prioritization, aoe - prioritize targets affected with subspace rupture, use only if you have at least 5 valid targets
- orkulus starbase - debris vortex - enable autocast - cast if 10% armor is missing and was recently damaged
